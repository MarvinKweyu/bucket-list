Find the application at: https://bucket-list-iota.vercel.app/todos

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```bash
npm run lint
```
