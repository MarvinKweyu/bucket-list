## Production app

Find the application at: https://cocky-neumann-011e1d.netlify.app


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```bash
npm run lint
```
